
5. Viết dictionary comprehension merge nhiều dictionaries với xử lý conflict.

```python

    {"a": 1, "b": 2},
    {"b": 3, "c": 4},
    {"c": 5, "d": 6}
]
# Lấy giá trị cuối cùng khi key trùng
# Kết quả: {'a': 1, 'b': 3, 'c': 5, 'd': 6}

```
