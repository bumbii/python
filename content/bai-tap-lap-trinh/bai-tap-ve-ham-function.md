# Bài tập về Hàm (function)

1. Viết hàm `greet()` in ra `"Hello World"`.
2. Viết hàm `say_hello(name)` in ra `Hello, <name>.`

Ví dụ:

```python
say_hello('Bumbii Academy')

# Hello, Bumbii Academy.
```

3. Viết hàm `add(a, b)` trả về tổng hai số.&#x20;

Ví dụ:

```python
c = add(10, 5) # 15
```

4. Viết hàm `square(n)` trả về bình phương của một số.

Ví dụ:

```
a = square(2) # 4
b = square(3) # 9
```

5. Viết hàm `is_even(n)` trả về `True` nếu số chẵn, `False` nếu số lẻ.&#x20;

Ví dụ:

```
is_even(10) # True
is_even(9) # False
```

6. Viết hàm `introduce(name, age)` in ra lời giới thiệu tên, tuổi.

Ví dụ:

```
introduce("Peter", 20)
# I'm Peter, I'm 20 years old.
```

7. Viết hàm `calculate_area(length, width)` trả về diện tích hình chữ nhật.

Ví dụ:

```
S = calculate_area(10, 5) # 50
S = calculate_area(8, 4) # 32
```

8. Viết hàm `fahrenheit_to_celsius(F)` chuyển đổi nhiệt độ sang Celsius.

Công thức chuyển độ F sang độ C:&#x20;

```
C = (F - 32) * 5 / 9
```
